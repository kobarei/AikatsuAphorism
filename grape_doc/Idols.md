### Idols



#### GET /:version/idols

 Return all idols.

**Parameters:**


 - fields : Additional fields with comma-separated list (first_name, last_name, brand, messages)



#### GET /:version/idols/:id

 Return a idol.

**Parameters:**


 - id (required) : Idol id.

 - fields : Additional fields with comma-separated list (first_name, last_name, brand, messages)
